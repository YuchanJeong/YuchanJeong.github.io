코드 스플리팅 #

리액트에서 활용 가능한 여러 방식의 코드 스플리팅이 있지만 우선 회사의 다른 프로젝트에서 사용하고 있는 loadable component를 활용한 페이지(라우팅) 기반의 코드 스플리팅만 적용 했다.

  1. yarn add @loadable/component @types/loadable__component

  2. App.tsx

     import loadable from "@loadable/component";
 ...

 const ExamplePage = loadable(() => import("pages/ExamplePage"))
 ...

 function App() {
   return (
     <Router>
       <Routes>
         <Route path="/example" element={<ExamplePage />} />
         ...
         <Route path="*" element={<NotFound />} />
       </Routes>
     </Router>
   )
 }

*_(underscore)를 scss 파일 이름 앞에 붙일 경우 부분 파일로 취급해서 작업 시점에는 모듈화되어 있지만 실제로 사용되는 시점에는 main.scss 파일 하나에 작성된다.
